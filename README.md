# Student Management System API

## Overview
A simple ASP.NET Core Web API built using Clean Architecture principles
for managing students (CRUD operations).

## Technologies
- ASP.NET Core Web API
- Entity Framework Core (In-Memory Database)
- Clean Architecture
- Swagger / OpenAPI

## Features
- Create student
- Get all students
- Get student by ID
- Update student
- Delete student

## Project Structure
- StudentManagement.Api
- StudentManagement.Application
- StudentManagement.Domain
- StudentManagement.Infrastructure

## How to Run
1. Clone the repository
2. Open the solution in Visual Studio
3. Run the project
4. Open Swagger at `/swagger`
